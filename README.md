# MRD Tech Shop Website

A premium, modern website for MRD Tech Shop (Laptop Repair & Sales).

## Features
- **Modern Dark Theme**: Designed with a "Supiri" (Premium) tech aesthetic using neon cyan and purple accents.
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop.
- **Glassmorphism**: Trendy frosted glass effects for cards and navigations.
- **Services Page**: Detailed list of repair services.
- **Products Page**: Showcase for laptops and accessories.
- **Contact Page**: Includes a contact form, phone numbers (0764385723 & 0759703718), and address (B.53 Meegahawela Road Koswatta).

## Technology
- HTML5
- CSS3
- JavaScript
- Tailwind CSS (via CDN - No installation required)
- FontAwesome Icons
- Google Fonts (Outfit)

## How to Run
1. Simply double-click on `index.html` to open the website in your web browser.
2. No installation or `npm` commands needed.

## Editing
- You can edit the text and images directly in the `.html` files using any text editor (like Notepad, VS Code, etc.).
